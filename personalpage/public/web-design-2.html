Assignment 1: Simple Page

This is a quiz that checks if the test taker is a musician or not in 5 simple questions. After the user completes the 5 questions, there are 3 different categories they can fall in. If the user gets less than 3 questions right, they aren't a musician at all. If the user gets exactly three right, they "may" be a musician. If the user gets more than 3 right, they are definitely a musician. Depending on the user's score, there will be 3 different memes that pop up, and match with their score result..
